# Retail Sales SQL Analysis

## Project Overview
This project performs end-to-end analysis of a retail sales dataset using SQL.
The objective is to extract meaningful business insights related to sales trends,
customer behavior, and product performance.

## Tools Used
- SQL (MySQL / PostgreSQL)
- GitHub
- CSV Dataset

## Dataset Description
The dataset contains retail transaction data including:
- Customer demographics
- Product categories
- Quantity and pricing
- Total sales amount

## Project Structure
Retail-Sales-SQL-Analysis/
├── README.md
├── data/
│   └── retail_sales.csv
├── sql/
│   ├── database_setup.sql
│   └── analysis_queries.sql

## How to Run This Project
1. Clone the repository
2. Run `database_setup.sql` to create and load the database
3. Execute queries from `analysis_queries.sql`
4. Modify queries to explore additional insights

## Analysis Performed
- Data exploration and cleaning
- Revenue analysis
- Product category performance
- Customer segmentation
- Time-based sales trends
- Ranking using window functions

## Key Insights
- Certain product categories generate the highest revenue
- Customers in specific age groups contribute more to sales
- Sales show clear monthly trends
